# TerraKur Bot

Node.js bot for MAX messenger with an Express API and mini-app tracker.

## Requirements

- Node.js 20+
- npm 10+

## Setup

1. Install dependencies:
   - `npm install`
2. Copy env template and fill values:
   - `copy .env.example .env` (Windows)
   - Set `BOT_TOKEN` and `MINI_APP_AUTH_SECRET`
3. (Optional) Rebuild routes GeoJSON:
   - `npm run build-data`

## Run

- `npm start` - start bot and API
- API health check: `GET /api/health`

## Security model for mini-app endpoints

Protected endpoints require:
- `chatId`
- mini-app auth token signed with `MINI_APP_AUTH_SECRET`

Token is generated by the bot and injected into mini-app links as `authToken`.
Mini-app sends token via `x-miniapp-auth` header.

## Quality tooling

- `npm run lint`
- `npm run format`
- `npm test`

CI (GitHub Actions) runs lint + tests on push and pull requests.
